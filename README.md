# Software Engineering Learning Repository

Welcome to my comprehensive learning repository for Software Engineering. This project serves as a structured collection of notes, code examples, and resources covering essential topics ranging from core Data Structures to High-Level System Design.

## 📚 Topics Covered

The repository is divided into four main modules:

### [1. Data Structures](./1.DataStructure)
Covers fundamental data structures and algorithms in depth, including:
- Arrays, Strings, and Hashing
- Linked Lists, Stacks, and Queues
- Trees (BST, Trie, Segment Trees, Heap)
- Graphs (BFS, DFS, Union-Find)
- Dynamic Programming
- Sliding Window, Two Pointers, Binary Search, Backtracking, and Recursion

### [2. Databases](./2.Databases)
Explores relational and non-relational database concepts:
- SQL basics and advanced queries
- NoSQL databases and Graph DBs
- Database Indexing, Partitioning, and Sharding
- Storage engines, cursors, and security design

### [3. LLD & Design Patterns](./3.LLD_DesignPatterns)
Focuses on Low-Level Design (LLD) principles:
- Object-Oriented Programming (OOPs)
- UML Diagrams and class relations
- Comprehensive Design Patterns

### [4. HLD & System Design](./4.HLD_SystemDesign)
Dives into High-Level Design (HLD) and building scalable systems:
- Networking and Communication
- Software Architecture, Web, and Hardware components
- Load balancing, Caching, Microservices, and Databases
- Scalability, Security, and Testing

## 🚀 Getting Started

This repository primarily uses **Jupyter Notebooks** (`.ipynb`). To view or execute these files locally:

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd Learning_2
   ```
2. Set up a Python virtual environment (optional but recommended) and install Jupyter:
   ```bash
   pip install jupyterlab
   ```
3. Run Jupyter Lab to browse the notebooks:
   ```bash
   jupyter lab
   ```

You can also view the notebooks natively in IDEs like **VS Code** with the Jupyter extension installed.
